## Sample DAX Measures (Copy & Paste)

Right-click on powerbi_sales_fact → New Measure:

```
Total Sales = SUM(powerbi_sales_fact[total_amount])
Total Orders = COUNTROWS(powerbi_sales_fact)
Avg Order Value = DIVIDE([Total Sales], [Total Orders])
Total Tax = SUM(powerbi_sales_fact[tax_amount])
Net Sales = SUM(powerbi_sales_fact[subtotal_amount])
Customer Count = DISTINCTCOUNT(powerbi_sales_fact[customer_id])

Sales MTD = 
CALCULATE(
    [Total Sales],
    DATESMTD(powerbi_sales_fact[order_date])
)

Sales YTD = 
CALCULATE(
    [Total Sales],
    DATESYTD(powerbi_sales_fact[order_date])
)

Top Customer Sales = 
CALCULATE(
    [Total Sales],
    TOPN(1, ALL(powerbi_customer_dim[customer_name]), [Total Sales])
)
```

## Format Numbers
Select each measure → Modeling tab → Format → Currency
